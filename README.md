## ✨ Features

- **Dark cyan & gold** color scheme – modern, professional, easy on the eyes
- **Fully responsive** – works on desktop, tablet, and mobile
- **Scroll‑triggered animations** – powered by GSAP ScrollTrigger
- **Interactive elements** – hover effects on cards, buttons, and links
- **Vanilla‑tilt 3D effect** on team member cards
- **Stat counters** – numbers animate into view when scrolling
- **Fake contact form** – shows a success alert (no backend needed)
- **WhatsApp floating button** – instant chat
- **No external dependencies** except GSAP and vanilla-tilt (loaded via CDN)
- **No custom cursor** – clean and accessible

---

## 🛠️ Tech Stack

- **HTML5** – semantic structure
- **CSS3** – custom styling, flex/grid, backdrop‑filter, animations
- **JavaScript (ES6)** – interactivity, GSAP, ScrollTrigger, vanilla-tilt
- **GSAP** – scroll animations & timeline effects
- **Vanilla Tilt** – 3D tilt on team cards

---

## 📦 Deployment (Netlify)

1. **Download** the `index.html` file from this repository.
2. Go to [Netlify](https://netlify.com) and sign up/log in.
3. Drag and drop the `index.html` file onto the Netlify dashboard.
4. Your site will be live in seconds at a random `*.netlify.app` URL.
5. (Optional) Change the site name in Netlify settings to something like `ad-dev-agency`.

> No build step, no backend, no configuration – just drag & drop!

---

## 🔧 Local Development

1. Clone or download the repository.
2. Open `index.html` in any modern browser.
3. Edit the file with your favorite code editor to customize content, colors, or links.
